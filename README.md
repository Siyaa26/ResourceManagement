<h1 align="center">🏫 Reserva Campus</h1>

<p align="center">
  <b>Campus Resource Booking System</b>
</p>

<p align="center">
  A full-stack web application for efficient scheduling and management of campus resources, events, and facilities.
</p>

<hr>

<h2>📌 Overview</h2>

<p>
Reserva Campus is a web-based resource management platform that enables students,
faculty, and administrators to book and manage campus facilities such as laboratories,
seminar halls, conference rooms, and academic equipment.
</p>

<h2>🚀 Features</h2>

<ul>
  <li>Role-Based Access Control (Student, Faculty, Administrator)</li>
  <li>Resource Booking & Management</li>
  <li>Event Creation & Scheduling</li>
  <li>Booking Conflict Detection</li>
  <li>Administrative Dashboard</li>
  <li>Interactive Campus Navigation</li>
  <li>Secure Authentication & Authorization</li>
  <li>Real-Time Resource Availability Tracking</li>
</ul>
