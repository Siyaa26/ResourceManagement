About Reserva

Reserva is a campus resource and event management platform that enables efficient booking of college infrastructure while improving visibility of campus events.

🎯 Our Purpose
Built for the Modern Campus

Universities and colleges have many shared resources — labs, conference rooms, seminar halls, projectors — but managing bookings for these is often done through manual WhatsApp messages, spreadsheets, or outdated paper forms. Reserva solves this with a clean, modern platform that streamlines the entire process.

✨ Features
What Reserva Offers
🖥️

Smart Resource Booking

Book labs, conference rooms, seminar halls, and projectors with real-time conflict detection. No double-bookings, ever.

📅

Event Management

Create and manage campus events. Link bookings directly to events and keep everything organized in one place.

🗺️

Campus Map

A visual, interactive map of all campus buildings and rooms. See availability at a glance and book with a single click.

🛡️

Admin Controls

Faculty and admins can approve or reject booking requests, create events, and add new resources to the system.

📊

Booking History

Every user can track all their bookings — pending, confirmed, or cancelled — with complete status transparency.

🌙

Dark / Light Mode

Use the system comfortably at any hour. Toggle between a clean light theme and an elegant dark mode instantly.
